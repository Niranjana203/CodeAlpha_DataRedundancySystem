# CodeAlpha_DataRedundancySystem# 🚀 CodeAlpha - Data Redundancy Removal System

This project was developed as part of my internship with [CodeAlpha](https://www.codealpha.tech). It is a Flask-based web application that integrates with Firebase Realtime Database to detect and prevent duplicate data entries in real time.

---

## 🔍 Features

- ✅ Real-time duplicate detection using Firebase
- ✅ Prevents redundant data from being stored
- ✅ Clean and responsive HTML form
- ✅ Cloud-based backend with secure integration
- ✅ Simple and scalable architecture

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Database:** Firebase Realtime Database
- **Frontend:** HTML, Jinja2
- **Tools:** Git, GitHub
